Spear Attacks

Ground Attacks

  • - Neutral Light: The user thrusts the spear forward, delivering three quick jabs at a distance. There’s a safe gap between the user and the hitbox, so enemies too close might avoid it.
  • - Side Light: The user swings the spear in a circular motion twice while stepping forward. Enemies hit are knocked into the air, setting up for follow-ups.
  • - Down Light: The user leaps diagonally forward using the spear. If it hits, a flurry of hits follows, ending in a knockback that can lead into combos.

Air Attacks

  • - Neutral Air: The user spins the spear in a full circle above and in front of them, striking all enemies caught in the arc.
  • - Side Air: The user spins and hurls the spear forward, pushing enemies back with forceful knockback.
  • - Down Air: The user jabs downward with the spear. On hit, the user bounces slightly. If done over the ground, it can also rebound.

Specials

  • - Recovery: The user twirls the spear rapidly while rising like a helicopter. Opponents caught by the spinning spear are launched away.
  • - Ground Pound: The user dives headfirst with the spear spinning below. Enemies struck by the rotating spear will be sent flying horizontally.
